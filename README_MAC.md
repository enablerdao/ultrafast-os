# 超高速OS プロジェクト (macOS対応版)

このプロジェクトは、高速起動と応答性を重視したオペレーティングシステムの概念実装です。1秒以内の起動時間と50ミリ秒以内の画面表示を目標としています。

## macOSでの実行方法

### 必要なツール
- Clang コンパイラ (macOSにデフォルトでインストール済み)
- Make (Command Line Tools for Xcodeの一部)

Command Line Toolsがインストールされていない場合は、以下のコマンドでインストールできます：
```bash
xcode-select --install
```

### コンパイルと実行

1. **リポジトリのクローン**:
```bash
git clone https://github.com/ユーザー名/リポジトリ名.git
cd リポジトリ名
```

2. **macOS用Makefileを使用してコンパイル**:
```bash
make -f Makefile.mac
```

3. **日本語版シミュレータの実行**:
```bash
make -f Makefile.mac run-jp
```

4. **英語版シミュレータの実行**:
```bash
make -f Makefile.mac run-en
```

## プロジェクトの構成

このプロジェクトは以下のファイルで構成されています：

1. **シミュレータ**:
   - `os_simulator_mac.c`: 日本語版OSシミュレータ (macOS対応)
   - `fast_os_simulation_mac.c`: 英語版OSシミュレータ (macOS対応)

2. **ビルドファイル**:
   - `Makefile.mac`: macOS用のMakefile

3. **理論的なOS実装**:
   - `bootloader.asm`: ブートローダーのアセンブリコード
   - `kernel_entry.asm`: カーネルエントリーポイントのアセンブリコード
   - `kernel.c`: カーネルのCコード
   - `kernel_64.c`: 64ビット用カーネルのCコード
   - `linker.ld`: リンカスクリプト

## 注意事項

- このプロジェクトは教育目的で作成されたコンセプト実装です。
- シミュレータは実際のOSの動作を模擬していますが、実際のOSではありません。
- 実際のOSコンポーネント（ブートローダー、カーネルなど）は、macOSでは直接ビルドして実行することはできません。

## トラブルシューティング

### 表示の問題
ターミナルによっては、色や特殊文字が正しく表示されない場合があります。その場合は、ターミナルの設定で「Unicode UTF-8」をサポートするフォントを選択してください。

### コンパイルエラー
コンパイルエラーが発生した場合は、Command Line Toolsが正しくインストールされているか確認してください。
